Markdown基本用法总结

June 23, 2021 406

标题

若一段文字被定义为标题，只要在这段文字前加#号即可。
为规范书写格式，#号后应当添加一个空格。
几个#号即代表几级标题。

引用

引用的文字段落以 > 号开头，如：

这是引用的内容

链接

将链接文字以方括号括起来，紧跟一个圆括号，圆括号内放入链接地址，如：
百度

图片

图片是一种特殊的链接，格式为 !+方括号+圆括号 。
方括号内是说明文字，圆括号内为图片地址。图片地址需要用到图床。

粗体与斜体

使用一对星号包含的内容为斜体，而双星号则表示粗体，如：
斜体 粗体

表格

表格使用制表符 | - : 等来表示。

代码

代码内容使用反引号来包含，如：
print("Hello world!")

分割线

分割线只需要另起一行，然后使用三个星号即可，如：

其他

换行

在一行的结尾使用双空格。

空行

使用与前后分离的 nbsp 标记。

空格

使用 nbsp 标记。
