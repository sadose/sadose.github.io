Hello World

June 22, 2021 599

第一次使用 Hexo + GitHub Pages 来搭建博客，逐渐弄明白了使用 Hexo 创建博客的基本流程，现作记录如下，当作是博客版的“Hello world”吧。

1 安装

切换到目录：
cd d:/hexo
安装hexo：
npm install hexo-cli -g

2 新建博客

新建：
hexo init blog
切换到目录：
cd blog
安装博客：
npm install

3 博客操作

生成静态文件：
hexo generate 或 hexo g
启动本地web服务（http://localhost:8000/）
hexo server -p 8000 或 hexo s
部署到远端
hexo deploy 或 hexo d
新建文章
hexo new "postName"
新建页面
hexo new page "pageName"

4 主题

以主题“yilia”为例
安装新主题：
hexo clean
git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia
启用主题：
修改Hexo目录下的_config.yml配置文件中的theme属性，将其设置为yilia。
更新主题：
cd themes/yilia
git pull
hexo g

5 其他

查看本地环境：
hexo -v
